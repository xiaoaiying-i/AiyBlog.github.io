vuex

vuex是vue的一个插件。

  • github 站点: https://github.com/vuejs/vuex
  • 在线文档: https://vuex.vuejs.org/zh-cn/
  • 对 vue 应用中多个组件的共享状态进行集中式的管理(读/写)

安装：npm install vuex –save

状态自管理

vuex对象

  • state, //状态对象
  • getters // 包含多个getter计算属性函数的对象
  • actions, // 包含多个对应事件回调函数的对象
  • mutations, // 包含多个更新state函数的对象

vuex使用时有时会把每个对象独立成一个模块

vuex结构

store对象

  • 所有vuex管理的组件都多一个属性：$store, 它是一个store对象

  • 属性：

    • state 注册的state对象
    • getters 注册的getters对象

  • 方法：

    • dispatch(actionNmae,data) 分发调用action

执行顺序

  • state 管理属性状态

  • getters 管理计算属性

  • $store.dispatch 调用 action

  • action 调用 murations
