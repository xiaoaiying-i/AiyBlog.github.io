Swagger简介

前后端分离

前后端通过API进行交互，相对独立且松耦合

  • 前端 -> 前端控制层、视图层
  • 后端 -> 后端控制层、服务层、数据访问层

Swagger

  • 流行的API框架
  • Restful Api 文档在线自动生成器 => API 文档 与API 定义同步更新
  • 直接运行，在线测试API
  • 支持多种语言 （如：Java，PHP等）
  • 官网：https://swagger.io/

SpringBoot集成Swagger

使用Swagger要求：jdk 1.8 + 否则swagger2无法运行

导包

  • Springfox-swagger2
  • swagger-springmvc

1、新建一个SpringBoot-web项目
